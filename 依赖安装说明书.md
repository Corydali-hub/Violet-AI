# 堇言AI — 依赖与安装说明书

## 1. 最低环境要求

| 环境 | 最低版本 | 推荐版本 | 说明 |
|------|----------|----------|------|
| 操作系统 | Windows 10 | Windows 11 | NapCat 仅支持 Windows |
| Python | 3.10 | 3.11 / 3.12 | Bot 核心 + WebUI 后端 |
| Node.js | 18 LTS | 20 LTS | NapCat 运行时 + 前端构建 |
| npm | 9.x | 10.x | 随 Node.js 自带 |
| QQ | 最新正式版 | — | NapCat 需要 Hook QQ 进程 |

## 2. 外部服务

| 服务 | 用途 | 获取方式 |
|------|------|----------|
| DeepSeek API Key | LLM 对话 | https://platform.deepseek.com 免费注册，送额度 |

## 3. Python 依赖

### 3.1 Launcher（WebUI 后端）

文件：`launcher/requirements.txt`

| 包名 | 版本 | 用途 |
|------|------|------|
| fastapi | >=0.110 | REST API 框架 |
| uvicorn | >=0.27 | ASGI 服务器 |
| aiohttp | >=3.9 | 异步 HTTP / WebSocket 客户端 |
| python-dotenv | >=1.0 | 读取 .env 环境变量 |
| websockets | >=12 | WebSocket 协议支持 |
| psutil | >=5.9 | 进程管理（启停 NapCat / Bot） |
| python-multipart | >=0.0.9 | 文件上传（背景图） |

安装：
```powershell
pip install -r launcher/requirements.txt
```

### 3.2 qq-ghost-bot（Bot 核心）

文件：`qq-ghost-bot/requirements.txt`

| 包名 | 版本 | 用途 |
|------|------|------|
| aiohttp | >=3.9, <4 | 异步 WebSocket 连接 OneBot |
| openai | >=1.0 | 调用 DeepSeek API（兼容 OpenAI SDK） |
| python-dotenv | >=1.0 | 读取 .env 备用配置 |

安装：
```powershell
pip install -r qq-ghost-bot/requirements.txt
```

> 💡 一行装完：
> ```powershell
> pip install -r qq-ghost-bot/requirements.txt -r launcher/requirements.txt
> ```

## 4. Node.js 依赖

### 4.1 前端（仅构建时需要）

文件：`launcher/frontend/package.json`

| 包名 | 用途 |
|------|------|
| vue 3.4 | 前端框架 |
| vue-router 4.3 | 页面路由 |
| element-plus 2.7 | UI 组件库 |
| @element-plus/icons-vue | 图标库 |
| axios 1.6 | HTTP 请求 |
| pinia 2.1 | 状态管理 |
| vite 5.2 | 构建工具（devDependency） |
| typescript 5.4 | 类型检查（devDependency） |

安装 + 构建：
```powershell
cd launcher/frontend
npm install
npm run build
cd ..\..
```

### 4.2 NapCat（已内嵌，无需单独安装）

`NapCat.Shell/` 目录已包含完整 NapCat 运行时，Node.js 安装后即可直接启动。

## 5. 一键安装脚本

把下面内容保存为 `install.bat`，放在项目根目录，双击运行：

```bat
@echo off
chcp 65001 >nul
cd /d "%~dp0"

echo ============================================
echo   堇言AI - 依赖安装
echo ============================================
echo.

echo [1/3] 安装 Python 依赖...
pip install -r qq-ghost-bot/requirements.txt -r launcher/requirements.txt
if errorlevel 1 (
    echo [ERROR] Python 依赖安装失败，请检查是否已安装 Python 3.10+
    pause
    exit /b
)
echo [ OK ] Python 依赖安装完成

echo [2/3] 安装前端依赖...
cd launcher\frontend
call npm install
if errorlevel 1 (
    echo [ERROR] npm install 失败，请检查是否已安装 Node.js 18+
    pause
    exit /b
)
echo [ OK ] npm install 完成

echo [3/3] 构建前端...
call npm run build
if errorlevel 1 (
    echo [ERROR] 前端构建失败
    pause
    exit /b
)
cd ..\..
echo [ OK ] 前端构建完成

echo.
echo ============================================
echo   安装完成！
echo   双击 start.bat 启动项目
echo ============================================
pause
```

## 6. 常见安装问题

### pip 安装报错 / 速度慢

使用清华镜像：
```powershell
pip install -r launcher/requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### npm install 报错 / 速度慢

使用淘宝镜像：
```powershell
npm config set registry https://registry.npmmirror.com
npm install
```

### 提示 "python 不是内部命令"

Python 未添加到 PATH。

- 安装时勾选 "Add Python to PATH"
- 或手动添加 Python 安装目录到系统环境变量

### 提示 "npm 不是内部命令"

Node.js 未安装或未添加到 PATH。

- 从 https://nodejs.org 下载 LTS 版本安装
- 安装时默认会添加到 PATH

### NapCat 启动无反应

- 必须以**管理员身份**运行（NapCat 需要 DLL 注入权限）
- 确保 QQ 已安装且已登录过一次
- 首次扫码可能失败，关闭后扫第二次即可

### DeepSeek API 调用失败

- 确认 API Key 已正确填入（WebUI 设置页 → API Key）
- 检查账户余额：https://platform.deepseek.com
- 确认网络能访问 `https://api.deepseek.com`
